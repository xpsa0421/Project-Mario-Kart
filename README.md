# Project Mario Kart
A single-player Mario Kart recreated with Unreal Engine 5, purely blueprint-based.

<br />

### Features
* Ray tracing for the kart
* Drive, mini-jump, boost, honk, drift
* Particle effects
* Lightings
* UI made with Widget Blueprints
